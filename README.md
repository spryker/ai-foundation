# AiFoundation Module
[![Latest Stable Version](https://poser.pugx.org/spryker/ai-foundation/v/stable.svg)](https://packagist.org/packages/spryker/ai-foundation)
[![Minimum PHP Version](https://img.shields.io/badge/php-%3E%3D%208.3-8892BF.svg)](https://php.net/)

AiFoundation is a module for Spryker Ecosystem to provide foundation functionality for AI providers interactions.

## Installation

```
composer require spryker/ai-foundation
```
